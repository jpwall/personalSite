Stock

Stock is a stalled project that I am working on which is intended to be a go-to program that does everything stock related. Currently it’s capabilities include calculating the profit of a buy/sell stock trade. This concept was transferred from StockCalc. Another feature of this program is the capability of automatically selling once it reaches a certain price point. The program keeps checking the price and when it is greater than or equal to the specified sell price, it stops and shows that price.

Stockapp.js

This was another stock-oriented program that I made shortly after StockCalc. The main intention of this program though was to return the current price of a stock given the ticker symbol. This worked well but has since been replaced by stock, which is listed above. Most of the code from Stockapp.js was simply copied and pasted into stock.

StockCalc

This was a program that I created to calculate the profits from a stock given the buy price, sell price, number of shares, and fee. This was a rather simple program written in C++ and was mostly a small program to help me in doing the calculations because it was annoying to calculate them by hand every single time. This has since been replaced with stock. It has all the same functionality and more!

Desktop

In the winter of 2011 when I was in 6th grade, I really wanted to build a desktop. At the time I was planning on primarily using it for gaming but that changed relatively quickly. After a year or two of getting some use out of it with games, I got interested in GNU/Linux and bought a seperate hard drive to put Ubuntu on and test out. This sparked a trend of me using one Linux distribution for a year or so then switching to another. I have so far tested out Ubuntu, Ubuntu GNOME, Arch Linux, Debian testing and is currently running Debian stable. In the fall of 2015 I replaced both Arch and Windows with Debian 8 Jessie and quickly updated to testing. Shortly afterwards I reinstalled Debian stable to serve this website and to use as a web server.
